    1. What are the ways to integrate CSS as a web page?
    • CSS can be added to HTML documents in 3 ways: Inline - by using the style attribute inside HTML elements.
    • Internal - by using a <style> element in the <head> section. External - by using a <link> element to link to an external CSS file.
    • An inline style may be used to apply a unique style for a single element. To use inline styles, add the style attribute to the relevant element.
    1. What is embedded style sheets?
    • CSS stylesheets can be applied to an HTML document in 3 ways :- inline styles, embedded stylesheet, and external stylesheet.
    • Embedded style sheets appear in the style element in the head of an HTML page.
    • Embedded style sheets allow you to define styles for the whole HTML document in one place.
    • Place your CSS rules into an HTML document using the <style> element that is called embedded CSS. This tag is placed inside <head>
    1. What are the external style sheets?
    • External style sheets are CSS files that are linked to HTML files.
    • An external style sheet is a separate CSS file that can be accessed by creating a link within the head section of the webpage.
    • Multiple webpages can use the same link to access the stylesheet. The link to an external style sheet is placed within the head section of the page.
    • An external style sheet can be written in any text editor, and must be saved with a .css extension. The external .css file should not contain any HTML tags.
    1. What are the advantages and disadvantages of using external style sheets?
    • The advantages of using external style sheets are that they can be created once and then referenced by multiple HTML files, which can save a lot of time and effort. The disadvantages are that if the style sheet is not formatted correctly, it can cause the entire page to not display correctly.
    1. What is the meaning of the CSS selector?
    • A CSS selector is the first part of a CSS Rule.
    • It is a pattern of elements and other terms that tell the browser which HTML elements should be selected to have the CSS property values inside the rule applied to them.
    1. What are the media types allowed by CSS?
    • CSS supports different media types than HTML.
    • The most common media type is “all,” which applies to all media type devices.
    • However, all doesn’t apply to all media types all the time.
    • For example, if you use the all media type, screen and (min-width: 500px), a screen device with a width of 400px won’t match the media rule.
